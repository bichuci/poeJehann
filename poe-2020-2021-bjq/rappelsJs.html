Manipuler le DOM

Utiliser javascript sur le DOM a pour but de manipuler le HTML de la page. Il faut deux choses

  • Évenement : Déclencher une action
  • Action : Modifier, ajouter, supprimer, etc. des éléments du DOM

Pour faire ces actions, il faut que le DOM soit complétement chargé

  • Window : fenêtre
  • Document : Document html
